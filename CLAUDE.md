# Roon MCP — project notes

A minimal MCP server to control the Roon music player. See `README.md` for
the user-facing tool surface and configuration, and the
[GitHub issue tracker](https://github.com/bestimmaa/roon-mcp/issues) for
known bugs and planned work.

## References

- This is the Roon API we are using: https://github.com/RoonLabs/node-roon-api
  (plus the companion service modules `node-roon-api-browse`,
  `node-roon-api-transport`, and `node-roon-api-status`).
